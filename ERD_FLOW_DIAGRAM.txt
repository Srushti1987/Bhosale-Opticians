================================================================================
COMPLETE USER JOURNEY FLOW
================================================================================

STEP 1: REGISTRATION
    User → register.php → USERS table → Login

STEP 2: BROWSE PRODUCTS
    User → products.php → View PRODUCTS → See PRODUCT_IMAGES

STEP 3: ADD TO CART
    User → Clicks "Add to Cart" → CART table (user_id + product_id)

STEP 4: CHECKOUT
    User → cart-session.php → checkout-session.php → Creates ORDERS record

STEP 5: ORDER ITEMS
    System → Saves each cart item to ORDER_ITEMS table

STEP 6: BILL GENERATION
    System → Creates BILL record linked to order_id

STEP 7: FEEDBACK
    User → Redirected to feedback-form.php → FEEDBACK table

STEP 8: ADMIN REVIEW
    Admin → Views feedback → Responds → Updates FEEDBACK table


================================================================================
ADMIN JOURNEY FLOW
================================================================================

STEP 1: LOGIN
    Admin → login.php → ADMIN table → Dashboard

STEP 2: MANAGE PRODUCTS
    Admin → admin/products.php → Add/Edit PRODUCTS → Update PRODUCT_IMAGES

STEP 3: MANAGE STOCK
    Admin → Updates STOCK table → Sets reorder_level → Tracks updated_by

STEP 4: VIEW ORDERS
    Admin → admin/orders.php → View ORDERS → See ORDER_ITEMS → View BILL

STEP 5: MANAGE FEEDBACK
    Admin → admin/feedback.php → View FEEDBACK → Add admin_response

STEP 6: VIEW USERS
    Admin → admin/users.php → View all USERS → See feedback counts

================================================================================
DATA FLOW SEQUENCE
================================================================================

1. USER REGISTRATION → USERS table created
2. ADMIN ADDS PRODUCTS → PRODUCTS table populated
3. ADMIN ADDS IMAGES → PRODUCT_IMAGES table populated
4. ADMIN SETS STOCK → STOCK table populated
5. USER BROWSES → Reads from PRODUCTS + PRODUCT_IMAGES
6. USER ADDS TO CART → CART table entry created
7. USER CHECKS OUT → ORDERS table entry created
8. SYSTEM CREATES ORDER ITEMS → ORDER_ITEMS table populated
9. SYSTEM GENERATES BILL → BILL table entry created
10. USER GIVES FEEDBACK → FEEDBACK table entry created
11. ADMIN RESPONDS → FEEDBACK table updated
12. ADMIN UPDATES STOCK → STOCK table updated
