HOW TO USE:
========================================

1. IMPORT UPDATED DATABASE:
   - Go to phpMyAdmin
   - Drop old 'sunray_db' database
   - Import database.sql again
   - This will create products with gender field

2. USE NEW HOMEPAGE:
   - Rename index_updated.php to index.php
   OR
   - Access: http://localhost/project/index_updated.php

3. BROWSE BY GENDER:
   - Click Men/Women/Kids in navigation
   - Or use category cards on homepage
   - Filter products by gender

4. ADMIN CAN ADD PRODUCTS:
   - Login as admin
   - Go to Products
   - Add new product with gender selection
   - Price in Indian Rupees
