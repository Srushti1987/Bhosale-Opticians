Key Database Relationships:

  • Users → Orders: One-to-Many (One user can have multiple orders)
  • Users → Cart: One-to-Many (One user can have multiple cart items)
  • Users → Feedback: One-to-Many (One user can give multiple feedback)
  • Products → Product_Images: One-to-Many (One product can have multiple images)
  • Products → Stock: One-to-One (Each product has one stock record)
  • Products → Cart: One-to-Many (One product can be in multiple carts)
  • Orders → Order_Items: One-to-Many (One order can have multiple items)
  • Orders → Bills: One-to-One (Each order generates one bill)
  • Admins → Stock: One-to-Many (One admin can update multiple stock records)
  • Admins → Feedback: One-to-Many (One admin can respond to multiple feedback)

Important Business Rules:

  • Stock Management: Available quantity = Total quantity - Reserved quantity
  • Pricing: All prices are in Indian Rupees (₹)
  • GST Calculation: 18% GST applied on all products
  • Discount Range: Variable discount percentages (10%-30%)
  • Payment Method: Currently supports COD (Cash on Delivery) only
  • Stock Alert: Reorder alert when stock ≤ reorder_level (default: 10)
  • Password Reset: Tokens expire after 1 hour
  • Feedback Rating: Must be between 1-5 stars
  • Order Status Flow: pending → confirmed → processing → shipped → delivered
  • User Types: Separate tables for customers (users) and administrators (admins)

Database Indexes for Performance:

  • idx_users_email: Fast user login lookup
  • idx_admins_email: Fast admin login lookup
  • idx_products_category: Product filtering by category
  • idx_products_gender: Product filtering by gender
  • idx_products_status: Active product queries
  • idx_stock_product: Stock lookup by product
  • idx_cart_user: User cart retrieval
  • idx_orders_user: User order history
  • idx_orders_status: Order status filtering
  • idx_order_items_order: Order item lookup
  • idx_feedback_user: User feedback lookup
  • idx_feedback_product: Product feedback lookup
