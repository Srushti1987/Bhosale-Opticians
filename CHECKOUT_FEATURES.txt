========================================
CHECKOUT FLOW:
========================================

1. USER ADDS PRODUCTS TO CART
   - Browse products
   - Click "Add to Cart"
   - Items stored in localStorage

2. VIEW CART
   - Go to cart.php
   - Review items
   - Update quantities
   - Remove items
   - See subtotal

3. PROCEED TO CHECKOUT
   - Click "Proceed to Checkout"
   - Must be logged in (redirects if not)
   - Fill shipping information:
     * Name, Email, Phone
     * Address, City, State, Pincode
     * Payment method (COD/Online)

4. PLACE ORDER
   - Order saved to database
   - Order items linked to order
   - Redirects to bill page

5. VIEW INVOICE
   - Professional invoice displayed
   - Order number generated
   - Item-wise breakdown
   - GST calculation (18%)
   - Print or download option
   - Cart automatically cleared
