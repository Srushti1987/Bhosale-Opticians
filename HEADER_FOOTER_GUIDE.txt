HOW TO USE IN YOUR PAGES:
--------------------------

For ROOT LEVEL pages (index_updated.php, products.php, cart-session.php, etc.):

<?php
session_start();
require_once 'config.php';

// Set page variables
$page_title = "Home - Bhosale Opticians";
$active_page = "home"; // Options: home, products, cart
$base_url = ""; // Empty for root level

// Include header
include 'includes/header.php';
?>

<!-- Your page content here -->

<?php
// Include footer
include 'includes/footer.php';
?>


For ADMIN pages (admin/dashboard.php, admin/products.php, etc.):

<?php
